## About

A clean, single-page website built for **Cafe Sweets** — a cozy family-run cafe in Arcadia, California known for specialty lattes, handmade tea cakes, and hearty noodle bowls.

The site features a full-screen hero, menu showcase, photo gallery, customer reviews, business hours, embedded Google Maps, and a call-to-action section.

## Features

- **Single-page design** — fast, no build step, zero dependencies
- **Responsive layout** — adapts to desktop, tablet, and mobile with hamburger navigation
- **Smooth animations** — scroll-triggered fade-ins via Intersection Observer
- **Menu cards** — categorized grid layout with hover effects
- **Photo gallery** — masonry-style grid with zoom-on-hover transitions
- **Customer reviews** — curated 5-star testimonials with elegant card design
- **Business info** — hours table, embedded Google Maps, address and links
- **Performance** — lazy-loaded images, preconnected Google Fonts, minimal JS

## Tech Stack

| Technology | Usage |
|-----------|-------|
| HTML5 | Semantic markup, accessibility |

| JavaScript | Scroll effects, Intersection Observer for fade-in |
| Google Fonts | Cormorant Garamond (headings) + Inter (body) |

## Getting Started

No build tools or dependencies required. Just open the file in a browser:

```bash
git clone https://github.com/jasonyellow520/cafesweets.git
cd cafesweets
open index.html
```

Or deploy to any static hosting service (GitHub Pages, Netlify, Vercel, etc.).

### Deploy to GitHub Pages

1. Go to **Settings > Pages** in your repository
2. Set source to **Deploy from a branch**
3. Select **main** branch and **/ (root)** folder
4. Save — your site will be live at `https://jasonyellow520.github.io/cafesweets/`
